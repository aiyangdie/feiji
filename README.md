# 太空战机游戏

这是一个使用HTML5 Canvas和JavaScript开发的太空射击游戏。

## 项目结构

- `index.html` - 游戏的主页面
- `game.js` - 游戏的主要逻辑代码
- `style.css` - 游戏的样式文件
- `io/` - 存放游戏资源文件的目录

## 游戏特点

- 经典的太空射击游戏玩法
- 响应式设计，支持键盘和鼠标操作
- 实时分数显示
- 游戏暂停/继续功能
- 操作说明面板

## 操作说明

### 移动控制
- ↑ 或 W：向上移动
- ↓ 或 S：向下移动
- ← 或 A：向左移动
- → 或 D：向右移动

### 功能键
- 空格：开始/暂停/继续游戏
- ESC：关闭教程
- P：暂停/继续游戏
- R：重新开始游戏
- 鼠标点击"开始游戏"按钮

## 如何运行

1. 确保您的浏览器支持HTML5
2. 直接在浏览器中打开`index.html`文件
3. 点击"开始游戏"按钮或按空格键开始游戏
